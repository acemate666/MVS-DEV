# 验收指标与排障清单（工程口径）

本篇给出与两阶段结构匹配的验收指标与线上排障顺序，目标是让问题定位优先回到“时间基准/分段/噪声口径”，避免把症状误判为“动力学不够复杂”。

---

## 1. 分阶段验收口径

### 1.1 prior（无 post 点）

- 名义预测：不出现量级失配（避免米级灾难性偏差）
- 走廊覆盖率：例如 90% / 95%
- 走廊宽度：在覆盖率达标前提下尽可能小

建议优先验收“按平面输出”的走廊（拦截点/落点更贴近任务）。

### 1.2 posterior（有 post 点，典型 $N\le 5$）

- 误差随 $N$ 的收敛曲线：$N=1..5$ 分别统计
- 锁错率（若存在多候选）：
  - 以 MAP 分支或最小 $J^{post}$ 分支为决策，统计分支错误率
- 任务指标：
  - 拦截平面误差（位置 + 到达时间）
  - 落点误差（见 2）

---

## 2. 落点/触地的定义（口径必须一致）

落点（第二次触地）应定义为第二段轨迹与球心触地高度 $y=y_c$ 的**下降穿越**：

- $y_c = y_{ground}+r+b_y$
- 选择规则：$\tau>0$ 且 $\dot y(\tau)<0$

输出建议包含：

- 落点位置误差 $|e_{xz}|$（m）
- 落点时间误差 $|e_t|$（s）

---

## 3. 推荐的线上排障顺序（高性价比）

### 3.1 首先检查：$t_b$ 是否随 $N$ 漂移

- 若 $t_b$ 随新 post 点变化明显，优先怀疑：
  - 分段契约被破坏（post 点污染 prefit）
  - 反弹附近缺帧导致触地求根被推迟

修复路径：见 [`03_prefit_and_segmentation.md`](03_prefit_and_segmentation.md) 与 [`05_stage2_posterior.md`](05_stage2_posterior.md)（小窗联合估计 $t_b$）。

### 3.2 再检查：prefit 残差与有效点数

- `prefit_rms` 是否暴涨
- 有效点数是否不足
- 是否存在明显离群点未被剔除

### 3.3 再检查：后验 $v_y^+$ 是否异常大

- $v_y^+$ 异常大常提示 $\tau$ 被放大（本质仍是 $t_b$ 或时间戳口径问题）。

### 3.4 候选融合相关：权重是否过早塌缩

- $N=1..2$ 时若权重快速塌缩到单候选，且后续难以恢复，通常需要更保守的温度 $\beta_N$ 或更强的正则。

### 3.5 若接入 2D 观测：检查重投影统计

建议记录并回放：

- 每相机 / 全相机：重投影误差均值与 95% 分位
- 每帧有效相机数（门控后）

若重投影误差整体偏大：

- 优先怀疑投影口径不一致（去畸变/坐标系/时间同步）
- 或 2D-3D 关联错配

细节见 [`08_observation_2d.md`](08_observation_2d.md)。

## 4. 建议固化的诊断字段（便于复现）

- prefit：`tb/pb/v_minus`、`prefit_rms`、`valid`、`t_freeze`、`freeze_reason`
- stage1：候选参数定义、`weights_before`
- stage2：每候选 `J_post`（建议拆分 data/prior term）、`beta_N`、`weights_after`
- 像素域（若启用）：重投影误差统计、是否回退与原因码
