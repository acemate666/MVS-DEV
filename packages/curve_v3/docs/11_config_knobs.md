# 关键旋钮（配置视角的概念表）

本篇给出两阶段方案中常见的“必须可配置”的旋钮，并解释其物理/工程含义。

说明：

- 这里描述的是**概念旋钮**（工程上应以配置文件/配置结构体提供）。
- 默认值属于建议范围，需结合数据标定与上线风险偏好调整。

---

## 1. Prefit / 分段（时间基准优先）

| 概念旋钮 | 含义 | 调大倾向 | 调小倾向 |
| --- | --- | --- | --- |
| 触地球心高度 $y_c$ | $y_{ground}+r+b_y$ | 触地更早（同样数据下） | 触地更晚 |
| 窗口长度 | 用于 prefit 的点数/时间跨度 | 更平滑但模型失配风险 ↑ | 更敏捷但噪声敏感 ↑ |
| 去抖时间 $T_{down},T_{up}$ | 趋势反转确认门限 | 误触发更少 | 触发更敏感 |
| 近地容忍 $\epsilon_y$ | 近地条件容忍度 | 更易触发 | 更严格 |
| gap 判定倍数 | 缺帧缺口阈值（相对正常帧间隔） | 更容易认为“不可见” | 更保守 |
| gap-freeze 裕量 $\delta$ | $\hat t_b$ 是否落入 gap 的容忍 | 更容易冻结 | 更不容易冻结 |

prefit 的首要目标是：冻结后的 $t_b$ 随回放可复现，且不被 post 点污染。

---

## 2. Stage1 候选与走廊

| 概念旋钮 | 含义 | 影响 |
| --- | --- | --- |
| $e$ 分档 | 法向恢复系数候选范围与密度 | 覆盖率 vs 候选数 |
| $k_t$ 分档 | 切向缩放候选范围与密度 | 覆盖率 vs 候选数 |
| $\phi$ 分档 | 切向偏转候选 | 多峰覆盖 ↑，候选数 ↑ |
| 候选数 $M$ | $|e|\cdot|k_t|\cdot|\phi|$ | 算力线性增长 |
| corridor 表示法 | quantile / mixture | 多峰安全性与下游使用方式 |
| 输出方式 | 按时间 / 按平面 | 任务贴合度与算力 |

---

## 3. Stage2 few-shot 校正

| 概念旋钮 | 含义 | 建议 |
| --- | --- | --- |
| 参数维度 | `v_only` 或 `v+axz` | $N$ 小/噪声大时更偏向 `v_only` |
| 正则矩阵 $\Lambda$ | MAP 先验强度 | 对 $a_{xz}$ 子块应更强 |
| 观测权重 $W$ | 观测噪声口径 | 建议支持按轴/按点 |
| 温度 $\beta_N$ | 权重塌缩速度 | $N\le2$ 更保守 |
| 异常点门控 | residual gating / 两步重加权 | 强烈建议具备 |

---

## 4. 联合 $t_b$（增强项）

| 概念旋钮 | 含义 | 备注 |
| --- | --- | --- |
| 小窗半宽 $\Delta$ | $t_b$ 搜索范围 | 缺帧时显著收益 |
| 先验方差 $\sigma_{t_b}^2$ | 防止 $t_b$ 乱漂 | 与 prefit 置信度相关 |
| 触发策略 | always / auto / off | 建议 auto：仅低置信时启用 |

---

## 5. 像素域精修（2D 观测增强项）

| 概念旋钮 | 含义 | 备注 |
| --- | --- | --- |
| Top-K | 只对前 K 个候选做像素精修 | K=1 或 2 通常足够 |
| 迭代上限 | GN/LM 最大迭代次数 | 固定 1–2 次保证实时 |
| Huber 阈值 | 像素域鲁棒核阈值 | 与像素噪声口径一致 |
| 帧级门控 | 最小有效相机数 | 常取 2 |
